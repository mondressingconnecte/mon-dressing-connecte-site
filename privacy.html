📋 Politique de Confidentialité

Mon Dressing Connecté - Application Mode IA

📅 Dernière mise à jour : Juin 2025
📧 Contact : mondressingconnecte@gmail.com

🎯 1. Présentation de l'Application

Mon Dressing Connecté est une application mobile d'assistance vestimentaire qui utilise l'Intelligence Artificielle pour analyser votre garde-robe et vous proposer des tenues adaptées à la météo et aux tendances mode Pinterest.

📊 2. Données Collectées

Notre application collecte uniquement les données essentielles à son fonctionnement :

📷 Photos de Vêtements

  • Images uploadées volontairement par l'utilisateur
  • Analysées localement puis via IA (Azure Vision, AWS Rekognition, Google Vision)
  • Stockées de manière sécurisée sur Firebase/Firestore

⚙️ Préférences Utilisateur

  • Styles préférés (casual, chic, sport, etc.)
  • Localisation (pour les données météo uniquement)
  • Historique des tenues portées (anonymisé)

📱 Données d'Usage

  • Statistiques d'utilisation anonymisées
  • Performances de l'application
  • Erreurs techniques (sans données personnelles)

🔧 3. Utilisation des Données

Vos données sont utilisées exclusivement pour :

  • 🤖 Analyse IA : Reconnaissance automatique des vêtements, couleurs et styles
  • 🌡️ Recommandations : Suggestions de tenues selon la météo
  • 📌 Inspiration : Recherche de tendances mode via Pinterest API
  • 📊 Amélioration : Optimisation de l'expérience utilisateur

🔗 4. Partage avec des Tiers

Nous ne vendons ni ne partageons vos données personnelles. Les seuls partages sont techniques :

Services d'IA (Analyse d'Images)

  • Microsoft Azure Vision : Analyse des vêtements et couleurs
  • AWS Rekognition : Détection d'objets vestimentaires
  • Google Vision API : Classification des styles

Pinterest API

  • Recherche d'inspirations mode (requêtes anonymisées)
  • Aucune donnée personnelle transmise

🔒 5. Sécurité et Stockage

Nous appliquons les meilleures pratiques de sécurité :

  • 🔐 Chiffrement : Toutes les données sont chiffrées en transit et au repos
  • ☁️ Firebase : Hébergement sécurisé avec authentification
  • 🌍 Localisation : Serveurs basés en Europe (RGPD)
  • 🗑️ Suppression : Possibilité de supprimer vos données à tout moment

👤 6. Vos Droits (RGPD)

En accord avec le Règlement Général sur la Protection des Données, vous avez le droit de :

  • 📖 Accès : Consulter toutes vos données stockées
  • ✏️ Rectification : Modifier vos informations
  • 🗑️ Suppression : Effacer définitivement votre compte
  • 📋 Portabilité : Exporter vos données
  • ⏸️ Opposition : Refuser certains traitements

🍪 7. Cookies et Traceurs

L'application utilise uniquement des cookies techniques nécessaires au fonctionnement :

  • Session utilisateur (connexion)
  • Préférences d'affichage
  • Pas de cookies publicitaires ou de tracking

👶 8. Protection des Mineurs

Cette application est destinée aux utilisateurs de 13 ans et plus. Les données de mineurs nécessitent l'autorisation parentale.

📝 9. Modifications

Cette politique peut être mise à jour. Les utilisateurs seront notifiés via l'application en cas de changements importants.
